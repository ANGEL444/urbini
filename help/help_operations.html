Operations - creating, editing, alerting and subscribing
 
List Operations This menu is shown when you are viewing a resource list. (see below for the menu shown when viewing a single item)
Edit List This enables rapid updates on the list of resources shown. Once you have made all the changes you want on the list click save.
Add Select this option to add a new resource of the type you are viewing.
Watch You can get notifications of any change or update for any class of resources. All alerts notifies you of any change. Special notices allows you to specify exactly what you want to watch allowing very flexible and fine control.
Set start page You can make any resource list with any filter your start page when you log on. Note that if your browser shortcut is set to take you to another specific class of information your home page will not work. You will have to remove the request part from your url.
       
Item Operations This menu is shown when you are viewing a single item.
Edit You can modify the item here. The layout is exactly the same as when creating an item including drop down menus, calendars and text boxes
Invite This allows you to invite anyone to subscribe to this item. They will receive an email from which they can choose to subscribe.
Add Select this option to add a new resource of the type you are viewing.
Subscribe Similar to watch but this is to get notifications of any change or update for this specificresource. All alerts notifies you of any change. Special notices allows you to specify exactly what you want to be notified about.
Unsubscribe As it suggests, you can remove your subscription from this item. (note you cannot remove anyone else)
Show all resources This is a quick way to see the list of this class of items in case you got to this page whilst traversing through your dashboard.
Set start page Again, the same as for resource lists, you can make this item your start page when you log on. Note that if your browser shortcut is set to take you to another specific class of information your home page will not work. You will have to remove the request part from your url.
