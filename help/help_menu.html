Menus - the icons at the top of the screen contain all the menu links.
       
Resource Operations This menu relates to operations you carry out on either a list of items or a single item, such as editing, creating, alerting etc.
Classes This menu will list all the current classes of items available.
Email This icon allows you to email the current screen as html or Excel to anyone on the planet. Enter their full internet email address.
Schedule This icon is for creating scheduled reports and charts. You can have a daily/weekly/monthly report of any list or item you're interested in.
Printing This icon will give you a printer friendly view. It removes the menu's, filtering tool and your name to create a view that looks right when printed.
PDA Mode This icon is switches to PDA mode. It has the same functionality but has been optimised for small screens.
Meet Now This icon will appear when you view any resource allowing you to chat on-line with any user about that particular item and record the conversation history, just like IM. You do not need an IM application to use this feature.
